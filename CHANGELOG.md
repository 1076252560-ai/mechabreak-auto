# Changelog

## v1.0.4 (2025-06-02)

- ARM_THRESHOLD 提高到 0.90，修复蓄能射束炮误识别为蓄能爆破炮
- 卡片检测仅限框选区域内 3x2 等分
- 未配卡片/刷新区域时禁用开始，GUI 红字提示
- 刷新按钮仅用框选坐标，删除模板匹配

## v1.0.3 (2025-06-02)

- EXE 文件名带版本号
- 打包加入 `--uac-admin` 自动管理员提权

## v1.0.2 (2025-06-02)

- 刷新按钮优先使用框选坐标
- 更新默认配置为 v1.0.1

## v1.0.1 (2025-06-02)

- F8 全局热键停止
- `run.bat` 本地一键运行
- 默认配置使用用户实际设置

## v1.0.0 (2025-05-27)

- 模板匹配识别武装和已售状态
- 紫色检测 IV 级武装
- Win32 API 模拟键鼠（SendInput + keybd_event）
- tkinter GUI 置顶窗口
- 全选 / IV级 / 按名称三种筛选模式
- 设置自动保存，下次启动恢复
- GitHub Actions 自动打包 Release
